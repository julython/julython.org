{% trans "How do I add a web hook for my project?" %}

{% trans "Github" %}

{% trans "If you host your project on github, follow the instructions below to setup a post recieve hook." %}

  1. {% trans "Go to your project on github." %}
  2. {% trans "Select the 'Admin' button at the top of the project dashboard." %}
  3. {% trans "Select 'Service Hooks' on the left menu." %}
  4. {% trans "In 'Available service hooks' select 'Webhook URLs'." %}
  5. {% trans "Enter the URL: 
    http://www.julython.org/api/v1/github
    " %}
  6. {% trans "Select 'Update Settings'" %}

{% trans "You can test it out by selecting 'WebHook URLs' again and choosing 'Test Hook'. Or you can try pushing a commit to your repository." %}

{% trans "Bitbucket" %}

{% trans "If you host your project on bitbucket, follow the instructions below to setup a post recieve hook." %}

  1. {% trans "Go to your project on bitbucket." %}
  2. {% trans "Select the 'Admin' admin tab on your project dashboard." %}
  3. {% trans "Select 'Services' on the left menu." %}
  4. {% trans "In the 'Select a service' dropdown select or type 'POST' then 'Add service'." %}
  5. {% trans "Enter the URL: 
    http://www.julython.org/api/v1/bitbucket
    " %}
  6. {% trans "Select 'Save Settings'" %}

{% trans "You can test it out by pushing a commit to your repository." %}
